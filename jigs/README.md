# Jigs

## Required Jigs

1. Temporary tuning cradle: repeatable support for first-strike and tuning-pass measurements.
2. Final foot placement template: repeatable isolation foot layout.
3. Kerf/tuning coupon holder: holds a small test strip safely for strike measurements.

## Cradle Requirements

- Supports the shell without touching tongues.
- Does not rattle at playing volume.
- Allows consistent microphone placement.
- Leaves the port accessible.
- Is marked with orientation so measurements are repeatable.
