# Data

Measured steel tongue drum results should be stored here once cutting and tuning begin.

## Expected Artifacts

- Rough-cut and post-deburr pitch logs.
- Kerf coupon measurements and process notes.
- Before/after tuning pass results.
- Final-mount and drift-check readings.
- Photos or screenshots that support `validation.csv` entries.
- The filled measurement path from `measurement-capture-plan.md`.

## Current Status

No measured prototype data exists yet. The repo is documentation-complete enough to guide a build, but this folder should remain empty until a real prototype is logged.
